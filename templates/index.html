Claude grades each exam against the rubric.

See more

Runs 5 parallel workers with built-in safeguards:

  • Borderline re-grade — scores within 1.5% of a letter grade cutoff are automatically graded twice and averaged
  • Feedback specificity — vague feedback ("Good work", "Incorrect") is detected and re-prompted for rubric-anchored detail
  • Deterministic output — temperature 0 for consistent scoring
  • 2x contrast enhancement on exam images for faint pencil
  • JSON retry (up to 2 attempts) catches malformed API responses
  • Anonymous IDs only — Claude receives the anon ID, answer pages, and rubric; cover pages and student names never leave this machine

Validate grading accuracy with an independent model.

See more

Re-grades a random sample using an independent model (Gemini cross-family or Opus within-family), then compares against production scores. Produces inter-rater reliability metrics:

  • Balanced stratified sampling — equal exams drawn from each version for fair coverage
  • Letter grade agreement, exact score match, within-1-point rates, and mean absolute error
  • QWK and ICC metrics when sample reaches 30+ exams
  • Downloadable PDF validation report for faculty review
  • Same privacy pipeline — anonymous IDs only, no student names sent to any API
